POST /api/thanks

send thanks to a recipient

Success Response Code: 201

Params

Param name Description
thanks
required
Value: Must be a Hash
thanks[body]
required

custom message for the thanks text


Value: Must be String
thanks[badge_id]
required

unique identifier of the Badge for the thanks


Value: Must be Integer
thanks[recipient_id]
optional

Deprecated: use recipient_ids. If this value is passed it will create a Thanks with a single recipient.


Value: Must be Integer
thanks[recipient_ids]
optional

unique identifiers of the Users to send thanks to


Value: Must be Array
thanks[in_reference_to_type]
optional

type of the reference, 'SocialObject' for a message and 'Comment' for a comment


Value: Must be String
thanks[in_reference_to_id]
optional

unique identifier of the reference


Value: Must be Integer

Response Fields

thanks[id] id of newly created thanks Integer
thanks[body] the custom message of thanks text String
thanks[badge] badge used for sending thanks Badge
thanks[user] information about the User that sent thanks User
thanks[recipient] Deprecated: use thanks[recipients]. Will contain the first value from thanks[recipients]. User
thanks[recipients] recipient information about the Users that recieved thanks Array[User]
thanks[annoucement] stream message broadcast for the sent thanks Message

Example Curl Commands

curl -X POST -v --basic -u "emily@socialcast.com:demo" "https://demo.socialcast.com/api/thanks.json" -H"Content-Type: application/json" -d'{"thanks":{"body":"You really helped me out!","badge_id":1,"recipient_ids":[21]}}'
curl -X POST -v --basic -u "emily@socialcast.com:demo" "https://demo.socialcast.com/api/thanks.json" -H"Content-Type: application/json" -d'{"thanks":{"body":"The two of you really helped me out!","badge_id":1,"recipient_ids":[21,26]}}'
curl -X POST -v --basic -u "emily@socialcast.com:demo" "https://demo.socialcast.com/api/thanks.json" -H"Content-Type: application/json" -d'{"thanks":{"body":"You really helped me out!","badge_id":1,"recipient_ids":[26],"in_reference_to_type":"SocialObject","in_reference_to_id":16}}'
