POST /api/users/:user_id/followers

the authenticated user will follow user_id

Also see Users#show.

Success Response Code: 201

Params

Param name Description
user_id
required

id of the user to be followed


Value: Must be Integer

Response Fields

user see Users#show for user details User

Example Curl Commands

curl -X POST -v --basic -u "emily@socialcast.com:demo" https://demo.socialcast.com/api/users/27/followers.json
