Topics

GET /api/topics/:topic_id
get a list of messages for a given topic

Success Response Code: 200

Params

Param name Description
topic
optional

topic to match


Value: Must be String

Response Fields

messages array of messages Array<Message>

Example Curl Commands

curl -X GET -v --basic -u "emily@socialcast.com:demo" https://demo.socialcast.com/api/topics/reference.json

Examples

GET /topics/sxsw
200

PUT /api/topics/:topic_id
update a topic

Success Response Code: 200

Params

Param name Description
topic_id
optional

id of topic to update


Value: Must be Integer
topic
optional

topic to update


Value: Must be String

Response Fields

topic[id] id of topic Integer
topic[name] name of topic String
topic[description] topic's description String
topic[url] url to interact with topic String
topic[avatars] set of images of various sizes to represent a topic Array<Avatars>

Example Curl Commands

curl -X PUT -v --basic -u "emily@socialcast.com:demo" -H"Content-Type: application/json" "https://demo.socialcast.com/api/topics/reference.json" -d'{"topic":{"description":"Keep this information handy! You might need it."}}'

GET /api/topics/search
get a list of topics that match the given search criteria

Success Response Code: 200

Params

Param name Description
q
optional

query string to match against. Matches against the topic name, title or description


Value: Must be String
name_prefix
optional

filters to only show topics that start with the requested name


Value: Must be String

Response Fields

topics topics array of topic entries Array<Topic>
topic[id] id of topic Integer
topic[name] name of topic String
topic[description] topic's description String
topic[url] url to interact with topic String
topic[avatars] set of images of various sizes to represent a topic Array<Avatars>

Examples

GET /topics/search.json?q=swsx
200
{
  "topics": []
}
