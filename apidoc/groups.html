GET /api/groups/:group_id/members
get a list of members of a group that the user has access to

Also see Users#show.

Success Response Code: 200

Params

Param name Description
group_id
required

id of group


Value: Must be Integer
role
optional

type of members to show (default=member)


Value: Must be one of: member, admin.
page
optional

page number to view (default=1)


Value: Must be Integer
per_page
optional

number of users to be returned per page (default=20)


Value: Must be Integer

Response Fields

users see Users#show for user details Array<User>

Example Curl Commands

curl -X GET -v --basic -u "emily@socialcast.com:demo" https://demo.socialcast.com/api/groups/2/members.json
