Success Response Code: 200

Response Fields

reach_extension see "POST /api/reach_extensions" for documentation on each response reach_extension

Example Curl Commands

curl -X GET -v --basic -u "adminuser@socialcast.com:adminuser" https://demo.socialcast.com/api/reach_extension/12.json
